# Film list

An app to store films you've watched or want to watch, with your comments, ratings, etc.

## Planned features

- a "have seen" list
- a "to watch" list
- custom tags
  - genres
  - rating
- custom reaction comments
- sorting and filtering by tags
- cross-device support
